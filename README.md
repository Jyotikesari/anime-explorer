🎌 Anime Explorer

A web application to browse, search, and filter anime using the Jikan API.

📌 Purpose
Anime Explorer lets users discover top-rated anime, search by title,
filter by genre, and sort by score or popularity — all in a clean,
responsive interface.

🌐 API Used
- [Jikan API](https://jikan.moe) (Unofficial MyAnimeList API)
- Endpoints: Top Anime, Search Anime, Anime by Genre
- No API key required

✨ Planned Features
- Browse top/popular anime
- Search anime by title
- Filter by genre (Action, Romance, Horror, etc.)
- Sort by score or popularity (ascending/descending)
- View anime details (poster, synopsis, episode count, rating)
- Save favorites using Local Storage
- Dark / Light mode toggle
- Responsive design for mobile, tablet, and desktop
  
🛠️ Technologies
- HTML, CSS, JavaScript (Vanilla)
- Jikan Public API (via fetch)
- Array HOFs: filter, sort, map, find

🚀 Setup
1. Clone the repository
2. Open `index.html` in your browser
3. No API key needed!
